## Getting Started

### Prerequisites

* Unity 2022.3.21f1

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lmxy0212/VRGaussianSplat.git
   ```

2. Open with Unity Hub
   ```sh
   project path: /VRGaussianSplat/VRGaussianSplat-Younique
   ```
